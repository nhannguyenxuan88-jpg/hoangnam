# Đặt hình ảnh placeholder tại đây

Để website hoạt động, bạn cần thêm ít nhất file `placeholder.jpg` vào folder này.

## Hình ảnh placeholder

Bạn có thể:
1. Tải ảnh placeholder miễn phí từ: https://placehold.co/800x800/png
2. Hoặc tạo ảnh đơn giản với text "No Image"
3. Hoặc dùng logo của shop

Đặt tên file: `placeholder.jpg`
