# Đặt hình ảnh bảo trì tại đây

## Cách thêm ảnh:

1. **Chụp ảnh xe trước/sau bảo trì**
   - Before: `{tên-xe}-before.jpg`
   - After: `{tên-xe}-after.jpg`
   - Main: `{tên-xe}-1.jpg`

2. **Ví dụ:**
   ```
   winner-engine-1.jpg      # Ảnh chính
   winner-before.jpg        # Trước sửa
   winner-after.jpg         # Sau sửa
   ```

3. **Cập nhật code trong `MaintenanceGallery.tsx`**
